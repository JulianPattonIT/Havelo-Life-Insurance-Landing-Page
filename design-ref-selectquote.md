# SelectQuote Design Reference — Key Observations

Studied: https://life.selectquote.com/termlife (2026-07-06)
Screenshots: `/home/team/shared/site/selectquote-ref.png`, `selectquote-ref2.png`

## Overall Structure

### Header
- Simple top bar with phone number prominently displayed: "1-855-241-3666"
- Logo (SelectQuote - colored) in header
- Clean, minimal navigation — no menu clutter

### Hero Section (Above the Fold)
- **Headline**: "Secure Rates Now" (H1) — short, action-oriented
- **Subheadline**: "Instantly compare rates from top carriers" (H2) — value proposition
- **Form**: Single input — ZIP code textbox + "Get a Free Quote" button
  - Minimal friction: only ask for ZIP code upfront
  - Button is disabled until ZIP is entered
- **Trust Signals**: Logo row below the form showing carrier logos (Prudential, Mutual of Omaha, Pacific Life, Symetra)
  - Establishes credibility through recognizable brand names

### Content Sections
- "Why buy life insurance?" — educational content section
- "SelectQuote Customer Reviews" — Trustpilot reviews carousel (social proof)
- "We do the shopping. You do the saving." — bottom CTA section with same ZIP form

### Footer
- Logo, social media links (Facebook, Twitter, LinkedIn, YouTube)
- Legal links: Privacy Policy, Legal, Disclosures, Accessibility
- BBB accreditation badge
- Network Solutions badge

## Design Patterns Noted

1. **Minimal form fields** — only ZIP code on first interaction. Collect more info later in funnel.
2. **Phone number prominently displayed** — builds trust, gives alternative conversion path
3. **Carrier logos as trust signals** — recognizable brand names validate the service
4. **Social proof** — Trustpilot reviews with detailed customer testimonials
5. **Multiple CTA touchpoints** — hero form + bottom-of-page form
6. **Clean whitespace** — spacious layout, not crowded
7. **Deep blue/navy** color scheme — trust and professionalism

## Key Takeaways for InsureLeads Redesign

- Consider a 2-step form: ZIP first, then collect more details
- Add carrier/partner logos for trust
- Display phone number prominently
- Add social proof section (reviews, testimonials)
- Keep the design clean and spacious
- Use deep navy + blue color scheme
